# One2Car-Scraper
Python script for scraping cars' data on https://www.one2car.com

## Prerequisites
- Python
- Libraries   
  - Pandas
  - BeautifulSoup

## Getting Start

1. Clone the repository

```sh
git clone https://github.com/pholawat-tle/One2Car-Scraper
```

2. Navigate to the project

```sh
cd One2Car-Scraper
```

3. Run the script

```sh
python revo.py
```
